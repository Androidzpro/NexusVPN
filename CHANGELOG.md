# 📦 CHANGELOG - NEXUSVPN PRO

## 🔥 v4.0 (2026-03-17)
### ✨ Nuevas características
- **4403 líneas de código** - Script completamente reescrito
- **UDP Custom** - Rango completo 1-65535 configurable
- **BadVPN dinámico** - Puertos configurables en tiempo real
- **Hysteria2** - Obfuscación salamander/random
- **WireGuard + AmneziaWG** - Soporte para ambos protocolos
- **IKEv2** - Compatibilidad nativa con iOS/Android
- **OpenVPN mejorado** - Generador automático de perfiles
- **SlowDNS** - Soporte para dnstt con dominio personalizado
- **Bot de Telegram** - Gestión completa desde el móvil
- **Panel Web** - Interfaz gráfica en puerto 8080
- **Monitoreo en vivo** - IPs, país, tiempo de conexión
- **Sistema de keys** - Control de expiración, usuarios y GB
- **Backups automáticos** - Programación diaria
- **Firewall avanzado** - UFW + iptables + rate limiting
- **Multi-idioma** - Soporte para ES/EN/PT/FR

### 🐛 Correcciones
- Instalación silenciosa sin interrupciones
- Detección automática de arquitectura (x86_64/ARM64)
- Reintentos en descargas fallidas
- Manejo de errores mejorado
- Logs detallados en `/var/log/nexusvpn/`

### ⚠️ Cambios importantes
- Contraseña admin por defecto: `NexusAdmin2024` (cambiar)
- Versión estable recomendada para producción
